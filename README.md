# landing--page-
A professionally designed responsive landing page built using Next.js and Tailwind CSS. This project was created as part of the AutoBotWa frontend developer assignment. The UI layout and several components — including the Hero Section, Features Section, and Testimonials — were generated with Vercel v0 to speed up the development workflow. The codebase is modular, includes reusable components, and follows modern frontend best practices. The landing page includes:
• Hero section with CTA
• Features section (3 items)
• Testimonials section
• Pricing section
• Footer
The project is deployed on Vercel and fully optimized for desktop, tablet, and mobile screens.
